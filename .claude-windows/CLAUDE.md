# CLAUDE.md

## 基本方針

- シンプルな構成・処理を優先する。
- コメント、コミットメッセージなどには、日本語を使用する。

## コミット

- 変更単位は小さく、こまめにコミットする。
- 変更を反映するたびにコミットしてよい（指示不要）。

## エージェント活用方針

- **サブエージェント・エージェントチームを積極的に活用し、並列処理を最大化する。**
- 独立したタスク（調査、コード探索、レビュー、テスト実行など）は可能な限り複数のエージェントを同時に起動して並列で処理する。
- 単一のエージェントで逐次処理するより、複数エージェントへの分割・並列実行を常に優先する。
- コードベースの探索・調査には Explore エージェントを活用し、メインコンテキストの消費を抑える。
- PR レビューやコード品質チェックには専門エージェント（code-reviewer, code-simplifier, silent-failure-hunter 等）を並列で走らせる。
- バックグラウンド実行（`run_in_background`）を活用し、待ち時間を削減する。
